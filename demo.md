# Waypoint Demo Script & Features

This document outlines the features and script for the Waypoint demo.

## Features to Highlight

- [x] **Barge-in**: Interrupt Clara mid-sentence — shows real-time not turn-based.
- [x] **Wave animation**: Synced to both user speaking and Clara speaking.
- [x] **Card group history**: Pills at top of sidebar — navigate between past card groups.
- [x] **Text mode toggle**: Mutes mic + Clara voice for text-only — a key accessibility and flexibility feature.
- [x] **Panel resize slider**: Premium draggable handle for layout adjustment, making the app feel responsive and polished.
